# Agent Jeremy

Agent Jeremy is a 2D game written in C++ and SDL library.

Jeremy is name of the hero you control.

## Controls:
* Arrows - moving
* Space - shoot
* p - pause
* n - new game
* esc - quit game
* m - menu
* enter - confirm choice
